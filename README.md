# Neo Physis - Sección CTA + Formulario

Implementación aislada de una sección moderna, responsive y semántica para una landing page científica.

## Sustentación

### 1. Arquitectura CSS
- Se utilizó metodología BEM para mantener nombres de clases claros, escalables y fáciles de mantener.
- La separación entre estructura, estilos e interacción permite que la sección sea fácil de integrar en otras páginas.
- Se priorizó una estructura simple y predecible para evitar dependencias innecesarias.

### 2. Organización de estilos
- Los estilos se agruparon por componente: CTA, beneficios, formulario y estado visual.
- Se trabajó con variables CSS para centralizar colores, bordes, sombras, radios y tipografía.
- Las media queries se aplicaron con enfoque mobile first para mejorar la lectura y el mantenimiento.

### 3. Ventajas de la metodología aplicada
- Facilita el crecimiento del proyecto sin que el CSS se vuelva difícil de leer.
- Reduce la posibilidad de conflictos entre clases.
- Permite reutilizar componentes visuales en futuras secciones de la landing page.

### 4. Buenas prácticas de diseño aplicadas
- HTML5 semántico para mejorar accesibilidad y orden del contenido.
- Formularios con labels, placeholders, focus states y mensajes de estado visibles.
- Botones con efecto hover, glow y microinteracciones para reforzar la experiencia de uso.
- Animaciones suaves para aportar dinamismo sin afectar la legibilidad.

### 5. Justificación UX/UI
- **Paleta de colores:** se eligieron tonos fríos y tecnológicos como azul profundo, cian y verde agua para asociar la interfaz con ciencia e innovación.
- **Tipografía:** se usó una fuente limpia y profesional para asegurar lectura cómoda y una apariencia moderna.
- **Jerarquía visual:** el título, el texto persuasivo, los beneficios y el formulario guían la atención de forma natural.
- **Accesibilidad básica:** se incluyeron etiquetas, estados focus y feedback visual para facilitar el uso del formulario.
- **Flujo de navegación:** la sección lleva de la propuesta de valor al registro sin fricción.

### 6. Aporte de cada estudiante
- Cada integrante puede explicar una parte específica del desarrollo: estructura HTML, diseño visual, responsive, validación y animaciones.
- Esto muestra trabajo colaborativo y dominio del componente desde distintas perspectivas técnicas.

### 7. Mensaje de cierre para exponer
- Neo Physis busca conectar investigación, divulgación y colaboración científica en una interfaz moderna, clara y profesional.
- La sección CTA + formulario está pensada para convertir interés en participación activa dentro de la comunidad.

## Archivos
- `index.html`: estructura semántica de la sección
- `styles.css`: estilos BEM, responsive, glassmorphism suave y animaciones
- `script.js`: validación visual y microinteracciones del formulario

## Uso
Abre `index.html` en el navegador o integra el bloque `<section class="neo-physis-cta">` en tu landing page.
